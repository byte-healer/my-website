## Color Palette

- Dominant: #FCBA04 (Gold), #000000 (Black)
- Accent: #9649CB (Purple)
- Neutral: #FFFFFF (White), #D4D2D5 (Light Gray)

## Setup

1. Clone or download this repository
2. Customize the content in each HTML file
3. Update personal information, skills, timeline, and blog posts
4. Replace placeholder images and links
5. Update social media links in `contact.html`

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push all files to the repository
3. Go to Settings > Pages
4. Select the main branch as the source
5. Your site will be live at `https://yourusername.github.io/repository-name/`

## Customization

### Update Personal Information
- Edit `about.html` for your bio and skills
- Update `timeline.html` with your career/education history
- Modify `contact.html` with your contact details

### Add Blog Posts
- Edit `blogs.html` and add new blog card entries
- Each blog card follows the same structure

### Change Colors
- Edit CSS variables in `css/style.css` at the top of the file

### Contact Form
The contact form currently shows an alert. To make it functional:
- Use [Formspree](https://formspree.io/) - free form handling service
- Use [EmailJS](https://www.emailjs.com/) - send emails directly from JavaScript
- Use [Netlify Forms](https://www.netlify.com/products/forms/) - if deploying to Netlify
